# Plan of PBS
성경 66권 연구를 위한 도구.

## Background
2024년 상반기 LTC 교육을 받으면서 성경 연구(이하 PBS)의 중요성을 깨닫고, 한 해 몇권의 PBS를 할지 계획, 총 66권의 PBS 완수하기 위한 계획, 기존 PBS 저장/관리할 수 있는 영역의 부재 등 Plan of PBS의 필요성을 느낌

## Prerequisites
- Oracle Java (JDK) installed: JDK 21
- Spring Boot 3.x
- Gradle (newest version)
- PostgreSQL 14.12
